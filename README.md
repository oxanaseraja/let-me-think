# Чат-бот с когнитивной картой

Бот в два этапа: **интервью** (опросник + свободное описание предпочтений) → **чат**, в котором объяснения подстраиваются под твою когнитивную карту. В чате можно прикладывать файлы — бот объясняет их содержимое с учётом карты.

## Установка

```bash
cd "let me think"
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
# В .env укажи OPENAI_API_KEY (и при необходимости OPENAI_BASE_URL)
```

## Запуск

- **Первый раз (или перепройти опросник):**  
  `python main.py interview`
- **Только чат (если профиль уже есть):**  
  `python main.py chat`
- **Авто:** если профиля нет — запускается интервью, иначе чат:  
  `python main.py`  
  или  
  `python main.py auto`

## Опросник

В интервью — 16 вопросов по темам:

- порядок подачи (правила/примеры, общая картина/шаги);
- уровень абстракции (паттерны, инварианты, структуры vs конкретика);
- формат (списки, заголовки vs нарратив);
- термины и формальность;
- объём и повторения;
- аналогии и связи с уже изученным;
- ошибки и неопределённость;
- темп и проверка понимания;
- явные паттерны и определения.

В конце — **свободное поле**: своими словами описать, как тебе удобнее объясняют новое (что помогает, что мешает, какие приёмы заходят). Ответы опросника + этот текст отправляются в LLM, которая формирует краткий фрагмент «когнитивной карты» для системного промпта.

## Чат

- Ввод — как обычно; ответ бота строится с учётом сохранённой когнитивной карты.
- **Приложить файл к следующему сообщению:**  
  `/файл путь/к/файлу` или `/file путь/к/файлу`  
  Затем вводишь текст вопроса — бот видит и содержимое файла, и твой запрос.
- **Выход:** `/выход` или `/exit`.

Профиль хранится в `cognitive_profile.json` в каталоге проекта. Его можно править вручную (поле `system_prompt_fragment`) или пройти интервью заново.

## Структура проекта

- `main.py` — точка входа (interview / chat / auto).
- `questionnaire.py` — вопросы и варианты ответов.
- `profile.py` — загрузка/сохранение профиля, формирование текста для системного промпта.
- `interview.py` — фаза интервью, вызов LLM для сводки карты.
- `chat.py` — чат с картой и поддержкой прикрепления файла.
- `llm.py` — вызов OpenAI-совместимого API.
